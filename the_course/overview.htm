Overview: MarkLogic Data Integration

Length: 2 days

In this 2-day course, you integrate data from disparate systems to create a data hub in MarkLogic. First, you load data from a variety of sources that is in a variety of structures. Then you harmonize the data, while learning a number of techniques for doing so. Next you load and use triples, including for harmonization. Then you create templates, for use with SQL and Optic API queries. Optionally, you then provide SQL-based applications with access to your data hub. Next, while learning how to write Optic API queries, you stand up a 360° customer view application that uses your data hub. Finally, you learn how to use Entity Services to model entities and generate useful artifacts.

The prerequisite for this course is the MarkLogic Fundamentals course. In addition, recommended prerequisites include any of the MarkLogic Developer I courses (XQuery, Node.js, or Java), and the MarkLogic Semantics course.

For live training online, you will want to have two computers (or one computer with two monitors) so that you can see what the instructor is doing on one monitor, while you are working on the second monitor.

For the self-paced version of this course, you will want to do the setup steps in the self-paced course videos to ensure that your computer will be able to run the course virtual machine (downloadable from MarkLogic University) locally.

This course was written on the CentOS platform, using a virtual machine from MarkLogic University (for running on a Windows, Macintosh, Linux, or Solaris host computer).

This course could also potentially be done without the course virtual machine, directly on a Linux computer (for setup, see See the Steps That Were Done), or on a Macintosh or Windows computer (with adjustments as needed, such as file paths, and MarkLogic installation steps). For a list of operating systems supported by MarkLogic, please see the MarkLogic Installation Guide.
